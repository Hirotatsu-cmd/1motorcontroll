/////////////////////////////////////////////////////////////////////////////

  実装機能:

プロジェクトのメインファイル 1motorcontroll.c:

1 個の割り込みハンドラ:
          ファイル Interrupt1.c: irq = Pci(4423,3346), handler + thread, タイムアウト = WAIT_FOREVER

2 個のメールボックス/セマフォスレッド:
          ファイル Server1.c: prio = 160, data mailbox Server1
          ファイル Server2.c: prio = 160, data mailbox Server2

2 個のポーリングスレッド:
          ファイル Poll1.c: プライオリティ = 170, nucleus sleep 100 msec
          ファイル Poll2.c: プライオリティ = 170, nucleus sleep 100 msec
